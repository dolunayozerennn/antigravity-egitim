# 📸 Kapak Geri Bildirim Klasörü

Bu klasör, KULLANICI_ADI_BURAYA Reels Kapak projesinin ürettiği görselleri değerlendirmek için kullanılır.

## Klasör Yapısı

```
feedback/
├── begendigim/        → ✅ Beğendiğin kapak tasarımları
├── begenmediklerim/   → ❌ Beğenmediğin kapak tasarımları
└── README.md
```

## Nasıl Kullanılır

1. Daha önce üretilen kapak görsellerini ilgili klasöre sürükle-bırak
2. Antigravity'ye hangi görselleri neden beğendiğini / beğenmediğini söyle
3. AI, bu geri bildirimlere göre gelecek kapak üretimlerini iyileştirir

## İpuçları

- Mümkünse her görselin dosya adını değiştirmeden bırak (orijinal isim takip için önemli)
- Beğenmediğin görseller için kısa bir not belirtirsen (ör: "renk çok koyu", "yazı okunamıyor") çok daha iyi sonuç alırsın
