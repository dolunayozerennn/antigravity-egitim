MCP Sunucusu Nedir?

Yapay Zekanın dış dünyayla bağlantı kurmasını sağlayan Evrensel Prizler.

1. Yapay Zeka (Beyin)

Çok zekidir, konuşmayı anlar, plan yapar, kod yazar. Ancak elleri ve gözleri yoktur. Kendi başına bilgisayarınızdaki bir dosyaya bakamaz veya internetten veri çekemez.

Düşünme Yorumlama

2. MCP Sunucusu

Yapay zeka için boyut ve dil gözetmeksizin her şeye uyabilen Evrensel Bir Priz (Universal Adapter) gibidir. Yapay zekanın eline gerekli "alet çantasını" verir ve dış dünyayla güvenli iletişim kurmasını sağlar.

Çevirmen Güvenli Köprü

3. Dış Dünya (Araçlar)

Bilgisayarınızdaki klasörler, GitHub deponuz, veri tabanlarınız, web siteleri veya API servisleri. Yapay zekanın müdahale edeceği gerçek çalışma alanları.

Sistemler Dosyalar

Gerçek Dünyadan Örnekler

Antigravity asistanınızın kullandığı bazı araç kutuları

GitHub MCP

Yapay zekanın sanki bir yazılımcıymış gibi kod okumasını, Commit atmasını ve Pull Request açmasını sağlar.

FileSystem (Dosya) MCP

Bilgisayarınızdaki "Masaüstü/Antigravity" içindeki dosyaları okuyup, onlara yeni kod yazmasına imkan tanır.

PostgreSQL MCP

Veritabanına bağlanıp, tabloları incelemesine ve size SQL sorguları çalıştırmasına olanak verir.
