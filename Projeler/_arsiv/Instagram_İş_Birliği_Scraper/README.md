# 📸 Instagram İş Birliği Scraper

Instagram üzerinden marka iş birliği yapılacak influencer/creator bulma, iletişim bilgilerini toplama ve e-posta gönderim sistemi.

## 📁 İçerik

- `scrape_ai_creators.py` — AI içerik üreticilerini tarama
- `find_contacts.py` — İletişim bilgisi bulma (email, sosyal medya)
- `analyze_brands.py` — Marka analiz aracı
- `send_emails.py` — E-posta gönderim scripti
- `scrape_reels.py` — Reels verisi çekme
- `İletişim_Metni_Örnekleri/` — E-posta şablonları
- `data/` — Çıktı verileri
